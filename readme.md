# install

```
npm install
```

# config
1. rename _config.js to config.js
2. edit config.js file.
```
export default {
	subaccount : "enter ftx sub account name",
	key : "enter ftx api key",
	secret : "enter ftx api secret",
	port : 8081
}
```
